# ECG-Project
ECG project using AD8232 module with an Arduino UNO R3

## Hardware required  
Arduino UNO/NANO

AD8232 module

Electrode connector and 3 ECG electrodes 

5 Male-Female jumper wires 

## Software required 
Arduino IDE
